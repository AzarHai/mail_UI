FAQ

Чтобы отправить письмо, выполните следующие шаги:
  1. Нажмите кнопку "Создать письмо" в верхней панели.
  2. Введите адрес получателя (например: user@example.com).
  3. Заполните поле тема письма.
  4. Введите текст письма в большом текстовом поле.
  5. Нажмите кнопку "Отправить".

Используйте строку поиска для поиска письма:
  1. Перейдите в нужную папку (например, "Входящие" или "Отправленные").
  2. В верхней части экрана найдите строку поиска.
  3. Введите ключевые слова: тему письма, адрес отправителя или часть текста письма.
  4. Результаты отобразятся в списке ниже.

Удалённые письма можно найти следующим образом:
  1. Откройте левую боковую панель.
  2. Нажмите на папку "Удалённые".
  3. Здесь будут отображаться все письма, которые вы удалили вручную.

Да, для восстановления удалённого письма:
  1. Перейдите в папку "Удалённые".
  2. Откройте нужное письмо.
  3. Нажмите кнопку "Восстановить" или переместите письмо в другую папку.

Чтобы изменить пароль:
  1. Нажмите на иконку профиля в правом верхнем углу.
  2. Выберите "Изменить пароль".
  3. Введите старый пароль, затем новый.
  4. Подтвердите новый пароль и нажмите "Сохранить".

Чтобы выбрать стиль интерфейса:
  1. Нажмите на иконку настроек (шестерёнка) в правом верхнем углу.
  2. Выберите один из стилей: По умолчанию, Компактный, Обычный.
  3. Нажмите на нужный вариант, изменения применятся автоматически.

Чтобы выбрать цветовую тему:
  1. Откройте панель настроек.
  2. В разделе "Тема интерфейса" выберите один из вариантов:
    • Светлая
    • Тёмная
    • Жёлтая
    • Синяя
    • Зелёная
  3. Щелкните по миниатюре темы — она применится сразу.
